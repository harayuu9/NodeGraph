# 主な機能

NodeGraph エディタが提供する主要な機能について解説します。

## ↩️ Undo / Redo

NodeGraph は堅牢な Undo/Redo システムを備えています。
`UndoRedoManager` クラスがコマンドパターンに基づいて操作履歴を管理します。

- **サポートされる操作**:
    - ノードの追加・削除
    - ノードの移動
    - ポートの接続・切断
    - プロパティの変更
    - 複数選択状態での一括操作

## 💾 シリアライゼーション

グラフの状態は YAML 形式で保存・読み込みが可能です。

- **.graph.yml**: グラフの構造（ノード、プロパティ、接続）を保存します。
- **.layout.yml**: エディタ上のレイアウト情報（ノードの座標）を保存します。

この分離により、ヘッドレス環境（CLIなど）ではレイアウト情報を無視してグラフを実行することが可能です。

## 🖱️ インタラクション

Avalonia UI を活用した直感的な操作性を提供します。

- **パン**: マウスの中ボタン（またはスペースキー + 左ドラッグ）でキャンバスを移動。
- **ズーム**: マウスホイールで拡大縮小。
- **矩形選択**: 左ドラッグで複数のノードを選択。
- **接続**: ポートからドラッグして接続線を作成。型互換性のあるポートのみハイライトされます。

## ⚡ 実行ステータスの可視化

グラフの実行中、各ノードの状態が視覚的にフィードバックされます。

- **待機中**: デフォルト状態
- **実行中**: 黄色の枠線
- **完了**: 緑色のチェックマーク
- **エラー**: 赤色のエラーマーク（例外メッセージを表示）

## 📋 コピー & ペースト

選択したノード群をクリップボード経由でコピー＆ペーストできます。
YAML形式でクリップボードに格納されるため、テキストエディタに貼り付けて内容を確認・編集することも可能です。
